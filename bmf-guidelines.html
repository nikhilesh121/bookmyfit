Effects & Treatments
Three signature effects combine to create the BookMyFit visual identity: photo layering, aurora colour washes, and glassmorphism panels.

Do
Always place real photos as the absolute background layer first
Apply aurora as mix-blend-mode:color — never opaque colour overlay
Add backdrop-filter only to panels floating above photo content
Keep glass alpha between 0.08–0.14 to preserve transparency
Add 1px rgba(255,255,255,.10–.18) border on all glass surfaces
Use DM Sans 700 for all numbers, prices, and stat values
Don't
Don't use solid colour cards — always photo + aurora first
Don't set aurora alpha above 0.85 — it kills the photo underneath
Don't use Font Awesome or any icon font — inline SVG only
Don't use Inter, Roboto or system fonts for display text
Don't use white or light backgrounds on any screen
Don't use Playfair Display for numbers — it looks wrong
